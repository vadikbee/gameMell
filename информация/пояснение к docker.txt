1. Сервис app (бэкенд на PHP/Laravel)
Сборка: Использует Dockerfile из ./docker/php/Dockerfile

Код: Монтирует папку ./backend в контейнер

Порты: Доступен на localhost:8000

Команда: Запускает Laravel сервер php artisan serve

Переменные окружения:

DB_HOST=mysql - подключение к базе данных

ROUTE_GENERATOR_URL=http://route-generator:5000 - интеграция с микросервисом

2. Сервис nuxt (фронтенд на Nuxt.js/Vue.js)
Образ: Официальный Node.js 18

Код: Монтирует папку ./frontend в контейнер

Порты: Доступен на localhost:3000

Команда: Запуск в режиме разработки npm run dev

3. Сервис route-generator (микросервис на Node.js)
Образ: Node.js 18

Код: Монтирует папку ./route-generator

Порты: Доступен на localhost:5000

Команда: npm start (вероятно Express/Koa сервер)

4. Сервис mysql (база данных)
Образ: MySQL 8.0

Настройки:

Пароль root: secret

Создаёт БД cockroach

Хранение данных: Том mysql-data для сохранения данных между перезапусками

Порты: Доступна на localhost:3306

5. Объём данных mysql-data
Постоянное хранилище для базы данных
